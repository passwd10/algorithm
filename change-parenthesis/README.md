# [괄호변환](https://programmers.co.kr/learn/courses/30/lessons/60058)

## 1. 이해

- '('와 ')'의 개수가 같다면 균형잡힌 괄호 문자열
- '('와 ')'의 짝도 모두 맞다면 올바른 괄호 문자열

## 2. 계획

1. 입력받은 균형잡힌 문자열을 앞자리부터 읽는다
2. '('룰 민니면 + 1, ')'를 만나면 -1
3. 0이 되는 순간 해당 문자열을 자른다
4. 자른 문자열의 앞부분을 u에 저장, 뒷부분을 v에 저장한다
5. ...
