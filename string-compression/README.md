# [문자열 압축](https://programmers.co.kr/learn/courses/30/lessons/60057)

## 1. 이해

- 문자열을 앞에서 부터 잘라야한다.
- 가장 짧은 문자열의 길이를 반환하라

## 2. 계획

1. 문자열을 앞에서부터 특정 개수만큼 일정하게 잘라 배열에 넣는다.
2. 배열에서 앞뒤로 중복되는 원소를 제거하고 중복된 원소의 개수를 앞에 넣는다
3. 특정 개수를 1부터 문자열 전체길이까지로 잡고 위 연산을 반복하고 결과값의 길이를 구한다.
4. 가장 짧은 길이를 찾아 반환한다.
