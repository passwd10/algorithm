# 2020년 5월 6일 문제풀이

## [쇠막대기](https://programmers.co.kr/learn/courses/30/lessons/42585)

1. 이해

    - 쇠막대기는 자신보다 긴 쇠막대기 위에만 놓일 수 있따
    - 쇠막대기를 다른 쇠막대기 위에 놓는 경우 완전히 포함되도록 놓아야한다.
    - 끝점이 겹치면 안된다
    - 쇠막대기를 자르는 레이저는 최소 하나 존재
    - 레이저는 끝점과 겹치지 않는다

2. 계획

    - '('를 만나면 쇠막대기 배열에 넣는다
    - ')'를 만나면 가장 뒤의 '(' pop하고 배열의 길이를 answer에 더한다.
