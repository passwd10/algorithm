# PS with python

## 터미널 사용법

```bash
#파일 실행
python [filename]

#전체 테스트 실행
ptw

#특정파일 테스트 실행
ptw [filename.py]

#특정경로 테스트 실행
ptw [dir]
```

> pytest를 실행하면 해당 프로젝트의 *_test.py 파일 안에 있는 모든 test_* 테스트 함수를 확인하게 된다.
